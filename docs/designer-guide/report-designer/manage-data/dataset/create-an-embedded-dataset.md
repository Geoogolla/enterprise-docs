# Create an embedded dataset

This section guides you to create an embedded dataset in Bold Report Designer.

## Design query data

1. Click the `Data` icon in the configuration panel to launch a `Data` configuration.

   <img style="width:412px" src="/assets/on-premise/images/report-designer/manage-data/dataset/data-icon-configuration-panel.png" alt="Data icon configuration panel">

2. Click on the `New Data` button in `Data` panel.

   <img style="width:540px" src="/assets/on-premise/images/report-designer/manage-data/dataset/new-data-button.png" alt="New data button">

3. In the connection type panel, click the data source type that you want to connect. Here, `SQL` connection type is used to demonstrate.

   <img style="width:540px" src="/assets/on-premise/images/report-designer/manage-data/dataset/connection-types-panel.png" alt="Connection types panel">

4. In the new data source configuration panel, fill the server name and related details.

   <img style="width:540px" src="/assets/on-premise/images/report-designer/manage-data/dataset/new-data-source-connection-panel.png" alt="New data source connection panel">

5. Click on the `Connect` button, now the following view will be displayed.

   <img style="width:540px" src="/assets/on-premise/images/report-designer/manage-data/dataset/query-designer-full-view.png" alt="Query designer full view">

6. You can edit the name of the `Data` in the **Name** field that is available in toolbar pane.

   <img style="width:540px" src="/assets/on-premise/images/report-designer/manage-data/dataset/edit-dataset-name.png" alt="Edit dataset name">

7. You can drag the tables and views associated with the connected database from left pane. Dragged table or view from the left pane can be dropped into the center pane labeled with `Drag and Drop tables here` like below.

   <img style="width:540px" src="/assets/on-premise/images/report-designer/manage-data/dataset/drag-drop-table-in-query-designer.png" alt="Drag and drop query table to design area">

   After you drop the item into the center pane, it displays like below:

   <img style="width:540px" src="/assets/on-premise/images/report-designer/manage-data/dataset/drag-drop-table-in-query-designer-output.png" alt="Query designer with query table">

8. To search the table or view from the data collection, you can use the `Search` field in the left pane of the query designer.

   <img style="width:235px" src="/assets/on-premise/images/report-designer/manage-data/dataset/query-designer-table-search-pane.png" alt="Query designer table search pane">

## Save Data

1. Click on the `Finish` button in the tools pane.
<img style="width:540px" src="/assets/on-premise/images/report-designer/manage-data/dataset/finish-data-connection.png" alt="Finish data button">
2. Your dataset should now be listed in the `Data` panel like below.
<img style="width:540px" src="/assets/on-premise/images/report-designer/manage-data/dataset/data-list-view.png" alt="Data list view">
3. Expand the icon to view the data fields.
<img style="width:540px" src="/assets/on-premise/images/report-designer/manage-data/dataset/data-fields-list.png" alt="Data fields list">
