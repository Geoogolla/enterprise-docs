# User Management Server

The Bold User Management Server lets you manage the applications and its access efficiently with the support of SSO (Single Sign-On). Users can be efficiently organized into groups to map the structure of the small and large organizations accurately. The User Management Server provides REST API to manage the users, groups, and applications.
